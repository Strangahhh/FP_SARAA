Project SARAA

Project SARAA focuses on revolutionizing the creation of Self-Assessment Reports (SAR) by leveraging Large Language Model (LLM) technology. Built on a Django Framework-based web application, SARAA (Self-Assessment Report Analysis Assistant) is designed to streamline document analysis and support SAR development. The system accepts user documents, categorizes content, facilitates question and answer sessions about the content, and recommends topics that align with AUN-QA standards.

Deploy on Hugging Face and Ollama

The SARAA model is trained on datasets specifically related to AUN-QA standards and relevant fields. It is then deployed on Hugging Face and Ollama platforms, ensuring robust, scalable, and efficient performance.

Developed with Django

Built with the Django Framework, SARAA offers a reliable and scalable foundation for web applications. Django's features simplify the development and integration of complex document analysis tools.

AUN-QA Standards Integration

SARAA integrates AUN-QA standards into its analysis and recommendations, helping universities meet quality assurance criteria effectively and streamline the Self-Assessment Report (SAR) creation process.
